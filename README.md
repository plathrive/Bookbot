# Bookbot
Bookbot takes books in form of ```.txt``` file and process it into total words and counted characters inside it.  
This project comes from Boot.dev, that took place in the 3rd course of DevOps path. This is my first project that i build,
and Boot.dev courses encourage me to build my first project.
### How to use it?
Simply clone or download this repo, install python (if you haven't), open your terminal (cmd, powershell, or unix), and run:
```
python3 main.py books/<title.txt>
```
You can try the books inside the repo that the course has provided, or also any ```.txt``` file would work the same.
